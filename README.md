# Pet Profile Project

A modern, mobile-first, single-page pet profile designed for NFC tags.

## Features
-   **NFC Ready**: Designed to be the landing page when scanning a pet's NFC tag.
-   **Mobile First**: Optimized for phone screens.
-   **Emergency Focus**: Prominent contact buttons and lost message.
-   **Privacy**: configured with `noindex` meta tags.

## Structure
-   `index.html`: Entry point.
-   `app.js`: Main React container.
-   `components/`: UI sections (Header, PetInfo, Emergency, etc.).
